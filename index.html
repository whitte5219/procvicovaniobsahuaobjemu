Pomůcky

Tato část slouží k pochopení rozdílu mezi pojmy obvod, obsah, povrch a objem. Nejde o počítání, ale o význam pojmů.

Význam pojmů

Obvod (2D)
  • Obvod je délka čáry kolem rovinného útvaru.
  • Jednotky: cm, m, km (jednotky délky).
Obsah (2D)
  • Obsah je plocha uvnitř rovinného útvaru.
  • Jednotky: cm², m² (jednotky plochy).
Povrch (3D)
  • Povrch je součet ploch všech stěn tělesa.
  • Jednotky: cm², m² (jednotky plochy).
Objem (3D)
  • Objem je prostor uvnitř tělesa.
  • Jednotky: cm³, m³ (jednotky objemu).

Vzorce pro jednotlivé tvary

Čtverec
  • Obvod: obvod = 4 · a
  • Obsah: obsah = a²
Obdélník
  • Obvod: obvod = 2 · (a + b)
  • Obsah: obsah = a · b
Kruh
  • Obvod: obvod = 2 · π · r
  • Obsah: obsah = π · r²
  • Poznámka: π ≈ 3,14
Trojúhelník (pravouhlý)
  • Obsah: obsah = (a · b) / 2
  • Obvod: obvod = a + b + c, kde c = √(a² + b²)
Trojúhelník (nepravouhlý)
  • Obsah: obsah = (a · vₐ) / 2
  • Obvod: obvod = a + b + c
Krychle
  • Povrch: povrch = 6 · a²
  • Objem: objem = a³
Kvádr
  • Povrch: povrch = 2 · (a · b + a · c + b · c)
  • Objem: objem = a · b · c
